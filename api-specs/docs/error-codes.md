# 🚨 錯誤碼對照表（E 系列）

| 錯誤碼   | 模組 | 說明                           | 建議處理                          |
|----------|------|--------------------------------|-----------------------------------|
| E2001    | M1   | 輸入字數過短或為空              | 請提示使用者輸入完整描述內容      |
| E4001    | M1   | 無法比對符合的警訊樣式          | 可回覆「目前無相關警訊對應」       |
| E2001    | M2   | 輸入不足，無法判斷階段          | 提醒補充更多行為或症狀資訊         |
| E4001    | M3   | 無法分類情緒或行為類別          | 可建議使用者改用更具體描述         |

---

## 📌 命名規則說明：

- `E1xxx`：系統層級錯誤（尚未使用）
- `E2xxx`：輸入格式錯誤
- `E3xxx`：授權/權限問題（預留）
- `E4xxx`：分析無結果 / 比對失敗

---

## 📢 建議前端處理方式

- 若 `status == error`，請根據 `error.code` 顯示對應訊息
- 可將錯誤碼對應翻譯顯示文字放在前端 i18n 檔案中維護
